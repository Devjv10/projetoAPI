# Product Catalog API

API REST para catalogo de produtos usando Node.js, Express e TypeScript.

## Como rodar

```bash
npm install
npm run dev
```

## Endpoints

- `POST /products`
- `GET /products`
- `GET /products/:id`
- `PUT /products/:id`
- `DELETE /products/:id`

## Documentacao

Acesse:

- `http://localhost:3000/api-docs`
